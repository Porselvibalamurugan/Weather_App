# 🌦️ Weather App

A simple and responsive **Weather Forecast Application** built using **HTML, CSS, and JavaScript**. This app fetches real-time weather and forecast data using the **OpenWeatherMap API**.

## ✨ Features

- 🌍 Search weather by city name  
- 🌡️ Displays current weather details  
- 📅 5-day weather forecast  
- ⏰ Live date and time display  
- 📱 Responsive and clean UI  

## 🛠️ Technologies Used

- **HTML5**
- **CSS3**
- **JavaScript (ES6)**
- **OpenWeatherMap API**
- **Bootstrap Icons**



## 🎯 Purpose

This project demonstrates **API integration, asynchronous JavaScript (Fetch API), DOM manipulation, and responsive UI development**.
